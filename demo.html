forceCalendar Interface

Web Components Calendar Interface - Demo Page

Live Calendar Component

This is a live demo of the @forcecalendar/interface package.

Features:

  • Multiple views (Month, Week, Day)
  • Event management
  • Timezone support
  • Locale customization
  • Web Components based (framework agnostic)

Installation

Install via npm:

npm install @forcecalendar/interface

Or use via CDN:

<script type="module" src="https://unpkg.com/@forcecalendar/interface@latest"></script>
